infrastructure-by-story
=======================

**Local cucumber installation based on facter**

This local approach relies on the presence of
- cucumber and
- facter

in a way that the step definitions check for the presence of certain facter facts.


Dependencies
============
```
builder (3.1.4)
cucumber (1.2.1)
diff-lcs (1.2.0)
facter (1.6.17)
gherkin (2.11.5)
json (1.7.6)
rake (0.8.7)
```
